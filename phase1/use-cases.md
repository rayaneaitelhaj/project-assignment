# Use Cases

## Use Case 1: Log a Workout
- Actor: Gym User
- Trigger: User clicks "Add Workout"
- Flow:
  1. Clicks "Add Workout"
  2. Fills info
  3. Saves
  4. Confirmation shown

## Use Case 2: Send Message to Trainer
- Actor: User
- Trigger: Click "Chat"
- Flow:
  1. Opens chat
  2. Types message
  3. Sends
  4. Trainer replies
